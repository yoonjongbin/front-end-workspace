p 요소는 단을 구분할 때 사용하고
br 요소는 줄을 바꿀 때 사용한다.
그리고 공백은 한 개의 공백만을 표시하기 때문에 별도의       기호 구문을 기술해야 한다. 

pre 태그는 시작태그부터 종료태그까지 입력된 내용 그대로 표현하는 태그
줄 바꿈같은        여러 개의 공백을 바로 인식함

그 밖에 글자를 다루는 태그들

b : 글자를 굵게 표시하는 태그
strong : 글자를 굵게 표시하는 태그

i : 글자를 기울여서 표시하는 태그
em : 글자를 기울여서 표시하는 태그

u : 글자에 밑줄이 그어지는 태그
s : 글자에 취소선을 넣어주는 태그
mark : 글자에 형광펜 효과를 넣어주는 태그

small : 글자를 작게 표현하는 태그
기본 글자에 sub : 아래첨자를 나타내는 태그와 sup : 윗첨자를 나타내는 태그

다른 글자들 사이에서 q : 인용문구를 나타내는 태그 어떻게 들어가는지 확인

다른 글자들 사이에서

blockquote : 인용문구를 나타내는 태그
어떻게 들어가는지 확인
